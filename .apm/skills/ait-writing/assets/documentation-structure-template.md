## Близость к коду и данным

Проверь:

- какие документы должны лежать рядом с кодом;
- какие документы должны лежать рядом с данными;
- какие документы должны быть вынесены в общий раздел;
- какие материалы нельзя разносить по разным местам без причины.

## Перекрытие содержания

Проверь:

- где тема уже описана;
- какие документы могут начать дублировать друг друга;
- где повторение полезно для читателя;
- где повторение создаёт риск расхождения;
- какой документ считается основным для спорной темы.

Перед созданием нового файла отдельно проверь, какие соседние документы уже
закрывают эту тему и какая новая задача читателя остаётся у будущего документа.
