# Подключение ai-dev-team

Проект использует `ai-dev-team`.

## Корень навыков ai-dev-team

Используй первый доступный корень навыков с обязательными файлами:

- `.agents/skills`;
- `.claude/skills`.

Корень подходит, если в нём есть файлы:

- `ait-routing/SKILL.md`;
- `ait-routing/references/skill-index.md`;
- `ait-setup/SKILL.md`.

Если ни один корень не подходит, остановись и сообщи о блокере.

## Маршрут задачи

Перед предметной работой открой `ait-routing/SKILL.md` из выбранного корня
навыков, выбери режим и покажи маршрут:

```text
Режим менеджера: лёгкий|полный.
Причина: ...
Маршрут: ...
```

Если маршрут выбирает навык из индекса `ait-routing/references/skill-index.md`,
до исполнения открой соответствующий `<skill>/SKILL.md` из выбранного корня
навыков и применяй его правила. Одного упоминания навыка в маршруте
недостаточно. Если выбранный навык отсутствует в корне навыков, остановись и
сообщи о блокере.

Для запросов вида «Настрой проект», «Перенастрой проект», «Начни новый проект»,
«Инициализируй проект», «Подготовь новый репозиторий» и близких по смыслу
запросов обязательно применяй `ait-setup/SKILL.md` из выбранного корня навыков.
