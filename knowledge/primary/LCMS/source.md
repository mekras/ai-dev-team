# Источник: `LCMS`

## Статус

Источник принят в корпус знаний как внешний публичный веб-материал
с ограничением на хранение полного HTML в Git.

## Происхождение

- Сайт: `https://www.langchain.com/blog`
- Дата получения: `2026-06-05T12:58:20Z`
- Способ получения: HTTP GET с локальным сохранением raw HTML и заголовков
  ответа
- Страница источника:
  `https://www.langchain.com/blog/how-and-when-to-build-multi-agent-systems`
- Заголовок страницы: `How and when to build multi-agent systems`
- `Last-Modified` по ответу сервера: `Fri, 05 Jun 2026 01:07:42 GMT`

## Что сохранено

В состав источника включены следующие артефакты:

- passport: `knowledge/primary/LCMS/source.md`
- индекс: `knowledge/primary/LCMS/page-index.tsv`
- страница:
  `knowledge/primary/LCMS/pages/how-and-when-to-build-multi-agent-systems/index.html`
- заголовки ответа:
  `knowledge/primary/LCMS/pages/how-and-when-to-build-multi-agent-systems/response-headers.txt`

Снимок хранится локально с исключением `pages/` из Git через
`knowledge/primary/LCMS/.gitignore`.

## Ограничения

- Источник ограничен одной страницей блога LangChain.
- Снимок отражает состояние страницы на момент получения.
- Полный текст страницы не должен использоваться как единственная опора без
  проверки условий использования.
- Нормализованные представления и извлечённые утверждения должны строиться из
  сохранённых артефактов этого источника, а не из живой страницы.

## Условия использования

- На уровне сохранённого HTML-снимка явная открытая лицензия на переиспользование
  полного текста не зафиксирована.
- Уровень уверенности: средний.
- Хранение полного HTML-артефакта в Git публичного проекта не допускается без
  отдельного подтверждения условий.
- Для внутренних задач допустимо извлечение проверяемых утверждений на основе
  сохранённого снимка.
- Атрибуция и правовой статус: `knowledge/source-attribution.md`.

## Очистка перед сохранением

В заголовках ответа удалены значения `set-cookie` и служебная строка прокси
`HTTP/1.1 200 Connection established`.

## Назначение в проекте

Материал источника может использоваться как первичный для изучения:

- критериев, когда multi-agent системы оправданы, а когда нет;
- практик context engineering и управления сложностью агентных систем;
- подходов к повышению надёжности и воспроизводимости агентной оркестрации.
