# Источник: `RTZ`

## Статус

Источник принят в корпус знаний как локально сохранённая рабочая копия
методического пособия по разработке технического задания на программное
обеспечение с ограничением на публикацию полного текста в Git.

## Происхождение

- Название:
  `Программное обеспечение для компьютерных систем и сетей. Разработка технического задания`
- Авторы: В.И. Дужий, А.В. Волковой, А.А. Волковая, А.А. Галькевич, А.С. Годунов
- Тип источника: локальная PDF-книга
- Способ получения: по решению пользователя в проект скопирована локальная
  рабочая копия документа с хранением файла только в игнорируемом каталоге
  `knowledge/primary/RTZ/files/`
- Дата включения в корпус: `2026-06-23`

## Что сохранено

Для локальной обработки получены следующие артефакты:

- паспорт: `knowledge/primary/RTZ/source.md`
- индекс артефактов: `knowledge/primary/RTZ/page-index.tsv`
- PDF: `knowledge/primary/RTZ/files/po-ksis-tech-task.pdf`
- текстовая выгрузка: `knowledge/primary/RTZ/files/po-ksis-tech-task.txt`

В Git сохраняются только метаданные источника. Обзорный нормализованный слой и
извлечённые утверждения готовятся отдельным шагом и без крупных дословных
фрагментов книги.

## Условия использования

- Открытая лицензия на повторное использование полного текста не установлена.
- Авторы содержания указаны на титульной странице и не совпадают с полем «автор»
  в метаданных PDF: поле метаданных отражает того, кто изготовил файл, а не
  правообладателя содержания.
- Уровень уверенности: средний.
- Полный PDF и полная текстовая выгрузка не публикуются в Git публичного
  проекта.
- Допустимое использование в корпусе: локальный анализ, нормализация тем,
  извлечение собственных кратких утверждений и ссылок на артефакты.
- Ограничение: не публиковать полный текст и крупные дословные фрагменты без
  отдельного правового основания.
- Атрибуция и правовой статус: `knowledge/source-attribution.md`.

## Ограничения

- Это локальная рабочая копия пособия, а не ссылка на издательский сайт.
- Поле «автор» в метаданных PDF не подтверждает авторство или права на
  содержание; правовой статус остаётся неустановленным.
- Текстовая выгрузка может содержать дефекты переноса строк и распознавания.
- При спорных формулировках нужно сверяться с PDF, а не только с выгрузкой.
- Извлечённые утверждения нельзя считать решениями проекта без отдельного
  анализа влияния.

## Назначение в проекте

Источник может использоваться как первичный материал для:

- структуры и содержания технического задания на программное обеспечение по ГОСТ
  19.201-78 и смежным стандартам;
- правил формулирования требований к программе и программному изделию;
- развития навыков и методических документов, связанных с требованиями и
  проектной документацией.
