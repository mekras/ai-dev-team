# Извлечённые утверждения: `HWD`

## Основание

- Источник учёта: `knowledge/inventory/HWD.md`
- Нормализованный слой: `knowledge/normalized/HWD.md`
- Индекс первичных артефактов: `knowledge/primary/HWD/page-index.tsv`
- Первичный снимок:
  `knowledge/primary/HWD/287c3b86170f56b163dc65eeaaf3678c3cefb25c/pages`
- Атрибуция и правовой статус: `knowledge/source-attribution.md`

## Назначение

Этот документ фиксирует первые утверждения по источнику `HWD`, пригодные для
дальнейшей работы над каркасом.

Утверждения ниже не являются ещё требованиями, правилами или решениями для этого
репозитория. Они фиксируют наблюдаемое содержание и структуру источника с опорой
на конкретные файлы.

## Что агент уже может использовать

- как основание для роли и навыка технического писателя:
  - цель документации как помощь читателю;
  - ориентация на одну аудиторию;
  - связь качества документации со скоростью разработки и ошибками;
- как основание для проектирования структуры корпуса:
  - разделение на `theory`, `guides`, `references`;
  - парность «подробное руководство + краткий справочник»;
  - наличие отдельных входных путей для быстрого решения и для первого старта;
- как основание для рабочих процедур:
  - сначала определить аудиторию;
  - затем выбрать место размещения и формат;
  - для новой статьи отдельно определить требования и составить план;
- как ограничение для дальнейшей работы:
  - не извлекать готовые правила из guide-страниц, которые ещё содержат только
    `todo`;
  - при спорных темах перепроверять опору на `theory/*` и завершённые обзорные
    страницы, а не на пустые заготовки.

## Ограничения

- Утверждения выше уже охватывают не только структуру, но и часть смыслового
  содержания ключевых страниц.
- При этом покрытие ещё не полное: не все разделы и не все страницы источника
  разобраны на атомарные утверждения.
- Они не должны автоматически превращаться в решения для каркаса без отдельного
  анализа.
- По темам, где исходные guide-страницы ещё не заполнены содержанием, источник
  пока задаёт только направление, но не готовое правило.
