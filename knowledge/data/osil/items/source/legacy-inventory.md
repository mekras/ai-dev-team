# Карточка источника: `OSIL`

## Идентификатор

`OSIL`

## Статус

- Состояние учёта: учтён
- Состояние получения: получен и сохранён в корпусе
- Состояние пригодности: пригоден для дальнейшей работы с оговорками

## Что это за источник

Внешняя публичная страница Open Source Initiative со списком OSI Approved
Licenses.

## Почему источник относится к проекту

Источник нужен как справочная точка о том, какие лицензии прошли review process
OSI и как они категоризуются.

## Основание в корпусе

- Паспорт источника: `knowledge/primary/OSIL/source.md`
- Индекс страниц: `knowledge/primary/OSIL/page-index.tsv`
- HTML-снимок: `knowledge/primary/OSIL/pages/licenses/index.html`
- Заголовки ответа: `knowledge/primary/OSIL/pages/licenses/response-headers.txt`

## Происхождение

- Платформа: `https://opensource.org`
- URL: `https://opensource.org/licenses`
- Дата получения: `2026-06-05T13:54:25Z`

## Тип источника

- Тип материала: первичные данные
- Тип формы: веб-страница
- Тип происхождения: внешний публичный сайт

## Условия использования

- Явная открытая лицензия страницы не зафиксирована.
- Уровень уверенности: средний.
- Сырой HTML хранится локально; в Git сохраняются метаданные и производные
  описания.
- Атрибуция и правовой статус: `knowledge/source-attribution.md`.

## Что уже известно по источнику

- Страница определяет открытые лицензии через соответствие Open Source
  Definition.
- На странице есть ссылка на `license review process` и набор категорий для
  навигации по лицензиям.

## Что можно делать дальше

- использовать источник как справочный реестр одобренных лицензий;
- при необходимости запрашивать конкретные карточки лицензий по URL.
