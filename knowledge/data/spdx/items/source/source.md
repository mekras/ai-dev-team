# Источник: `SPDX`

## Статус

Источник принят в корпус знаний как справочный набор страниц экосистемы SPDX.

## Происхождение

- Основные платформы: `https://spdx.org`, `https://spdx.github.io`
- Дата получения: `2026-06-05`
- Способ получения: HTTP GET с сохранением HTML-снимков и заголовков ответа.

## Что сохранено

- паспорт: `knowledge/data/spdx/items/source/source.md`
- индекс: `knowledge/data/spdx/items/primary-spdx-page-index.tsv/page-index.tsv`
- страницы:
  - `licenses`
  - `exceptions-index`
  - `license-expressions`
  - `matching-guidelines`
- заголовки ответа для каждой страницы

## Условия использования

- Явное разрешение на хранение полного HTML в Git не зафиксировано.
- Уровень уверенности: средний.
- Для внутренней работы допустимы метаданные, хэши и производные описания.
- Атрибуция и правовой статус: `knowledge/source-attribution.md`.

## Назначение в проекте

Источник используется для идентификации лицензий, записи SPDX-выражений, учёта
исключений и проверки совпадения текстов лицензий.
