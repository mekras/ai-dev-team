# Карточка источника: `SPDX`

## Идентификатор

`SPDX`

## Статус

- Состояние учёта: учтён
- Состояние получения: получен и сохранён в корпусе
- Состояние пригодности: пригоден для дальнейшей работы с оговорками

## Что это за источник

Внешний публичный набор справочных страниц SPDX о списке лицензий, исключениях,
выражениях лицензий и правилах сопоставления текста лицензий.

## Почему источник относится к проекту

Источник нужен для идентификаторов лицензий, записи выражений SPDX, учёта
исключений и проверки соответствия текстов лицензий.

## Основание в корпусе

- Паспорт источника: `knowledge/primary/SPDX/source.md`
- Индекс страниц: `knowledge/primary/SPDX/page-index.tsv`
- Снимки страниц: `knowledge/primary/SPDX/pages/**`
- Заголовки HTTP-ответов: `knowledge/primary/SPDX/pages/**/response-headers.txt`

## Происхождение

- Платформа: `https://spdx.org` и `https://spdx.github.io`
- Включённые страницы:
  - `https://spdx.org/licenses/`
  - `https://spdx.org/licenses/exceptions-index.html`
  - `https://spdx.github.io/spdx-spec/latest/annexes/spdx-license-expressions/`
  - `https://spdx.github.io/spdx-spec/latest/annexes/license-matching-guidelines-and-templates/`
- Дата получения: `2026-06-05`

## Тип источника

- Тип материала: первичные данные
- Тип формы: веб-страницы
- Тип происхождения: внешний публичный сайт и спецификация

## Условия использования

- Открытая лицензия на полный HTML-снимок явно не подтверждена.
- Уровень уверенности: средний.
- В Git сохраняются метаданные и производные материалы; сырые страницы сохранены
  локально.
- Атрибуция и правовой статус: `knowledge/source-attribution.md`.

## Что уже известно по источнику

- Список лицензий SPDX описан как часть спецификации SPDX.
- Страницы связаны с исключениями, выражениями лицензий и правилами matching.

## Что можно делать дальше

- извлечь правила построения SPDX-выражений;
- использовать источник для выбора идентификаторов и проверки совпадения текстов
  лицензий.
