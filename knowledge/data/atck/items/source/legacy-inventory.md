# Карточка источника: `ATCK`

## Идентификатор

`ATCK`

## Статус

- Состояние учёта: учтён
- Состояние получения: `external_reference` — локальная копия не хранится
- Состояние пригодности: пригоден (получать релевантную технику по запросу)

## Что это за источник

`MITRE ATT&CK` — база знаний тактик, техник и процедур (TTP) злоумышленников по
доменам Enterprise, Mobile, ICS.

## Почему источник относится к проекту

Библиотека TTP для навыка `security-threat-modeling` (что может сделать
злоумышленник, операционные модели угроз) и контекст для `security-audit`.
