# Источник: `SMFR`

## Статус

Источник принят в корпус знаний как локально сохранённая рабочая копия книги по
описанию вариантов использования и функциональных требований с ограничением на
публикацию полного DjVu и полной текстовой выгрузки в Git.

- Автор: Алистер Коберн
- Английское издание: `2001`
- Русское издание: издательство `Лори`, `2002`
- ISBN оригинала: `0-201-70225-8`
- ISBN русского издания: `5-85582-152-8`
- Тип источника: локальная DjVu-книга
- Способ получения: по запросу пользователя в проект скопирована локальная
  рабочая копия книги с хранением файла только в игнорируемом каталоге
  `knowledge/primary/SMFR/files/`
- Дата включения в корпус: `2026-06-22`

## Что сохранено

Для локальной обработки получены следующие артефакты:

- паспорт: `knowledge/primary/SMFR/source.md`
- индекс артефактов: `knowledge/primary/SMFR/page-index.tsv`
- DjVu:
  `knowledge/primary/SMFR/files/sovremennye-metody-opisaniya-funkcionalnyh-trebovanii.djvu`
- текстовая выгрузка:
  `knowledge/primary/SMFR/files/sovremennye-metody-opisaniya-funkcionalnyh-trebovanii.txt`

В Git сохраняются только метаданные источника, обзорный нормализованный слой,
извлечённые утверждения и правовой статус. Полный DjVu и полная текстовая
выгрузка исключены из Git.

## Условия использования

- Открытая лицензия на повторное использование полного текста не установлена.
- Выходные данные документа проверены по локальной текстовой выгрузке.
- Уровень уверенности: средний.
- Полный DjVu и полная текстовая выгрузка не публикуются в Git публичного
  проекта.
- Допустимое использование в корпусе: локальная обработка, подготовка
  производных метаданных, обзорная нормализация и извлечение собственных кратких
  утверждений после нормализации.
- Ограничение: не публиковать полный текст и крупные дословные фрагменты без
  отдельного правового основания.
- Атрибуция и правовой статус: `knowledge/source-attribution.md`.

## Ограничения

- Это локальная рабочая копия книги, а не ссылка на издательский сайт.
- Текстовая выгрузка получена через `djvutxt` после установки `djvulibre` через
  Homebrew.
- Выгрузка содержит дефекты распознавания, переносов и отдельных символов.
- При спорных формулировках нужно сверяться с DjVu, а не только с текстовой
  выгрузкой.

## Назначение в проекте

Источник может использоваться как первичный материал для развития практик
выявления, анализа, спецификации, проверки и сопровождения вариантов
использования, сценариев и функциональных требований.
