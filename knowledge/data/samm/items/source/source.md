# Источник: `SAMM`

## Статус

Источник принят в корпус знаний как локально сохранённая копия основной модели
`OWASP SAMM v2.0` (Software Assurance Maturity Model). Документ распространяется
по лицензии `CC BY-SA 4.0`. Полный PDF и текстовая выгрузка хранятся локально и
исключены из Git ради опрятности репозитория; канонический адрес и лицензия
позволяют использовать материал шире.

## Происхождение

- Название: `OWASP SAMM v2.0 - Core Model Document`
- Издатель: OWASP Foundation
- Версия: `2.0`
- Год издания: `2020`
- Канонический адрес проекта: `https://owaspsamm.org/` и
  `https://owasp.org/www-project-samm/`

- Дата включения в корпус: `2026-06-25`

## Ограничения

- Текстовая выгрузка получена `pdftotext -layout`; таблицы уровней зрелости и
  потоков в выгрузке перемешаны; при ссылке на конкретную активность нужно
  сверяться с PDF.
- Модель описывает практики и уровни зрелости, а не конкретные инструменты.
- `OpenSAMM` (упоминается в исходных заметках) — предшественник; в корпус
  включена актуальная версия 2.0.
- Извлечённые утверждения нельзя считать решениями проекта без анализа влияния.

## Назначение в проекте

Источник может использоваться как первичный материал для:

- модели зрелости разработки ПО: пять бизнес-функций (Governance, Design,
  Implementation, Verification, Operations), по три практики безопасности в
  каждой, по три уровня зрелости и два потока в каждой практике;
- оценки и планирования программы безопасной разработки;
- усиления навыка `security-audit` (оценка зрелости, измеримые активности) и
  связи с `security-threat-modeling` (практика Threat Assessment).
