## Группы метрик (CVSS v4.0)

- Base — внутренние характеристики уязвимости, постоянные во времени.
- Threat — текущее состояние эксплуатируемости (наличие эксплойта).
- Environmental — факторы конкретного развёртывания организации.
- Supplemental — дополнительные контекстные атрибуты (не влияют на балл).

## Базовые метрики

- Эксплуатируемость: Attack Vector (AV), Attack Complexity (AC), Attack
  Requirements (AT), Privileges Required (PR), User Interaction (UI).
- Влияние: конфиденциальность/целостность/доступность уязвимой системы
  (VC/VI/VA) и последующих систем (SC/SI/SA).

## Нотация

- Балл помечается по использованным группам: `CVSS-B` (только Base), `CVSS-BT`
  (Base + Threat), `CVSS-BE` (Base + Environmental), `CVSS-BTE` (все три).
- Нотацию следует указывать везде, где приводится числовое значение CVSS.

## Замечания

- CVSS оценивает техническую серьёзность, а не риск для конкретной организации;
  приоритизация требует учёта контекста и данных об угрозах.
- Существуют разные версии (v3.1, v4.0) — указывать версию.
