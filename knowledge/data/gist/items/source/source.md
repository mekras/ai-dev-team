ограничением на хранение полного HTML в Git.

- URL: `https://gist.github.com/0xfauzi/7c8f65572930a21efa62623557d83f6e`

- passport: `knowledge/data/gist/items/source/source.md`

  `knowledge/data/gist/items/7c8f65572930a21efa62623557d83f6e/index.html`

`knowledge/data/gist/items/source/legacy-primary.gitignore`.

## Ограничения

- Источник зафиксирован одним Gist-компаньоном.
- Gist может обновляться; фиксируется текущая версия как raw snapshot.
- Снимок отражает состояние на момент получения.
- Полный текст страницы не должен использоваться как единственная опора без
  проверки условий использования.

## Условия использования

- Лицензионные условия для полного переиспользования текста в сохранённом HTML
  не зафиксированы явно.
- Уровень уверенности: средний.
- Хранение полного HTML-артефакта в Git публичного проекта не допускается без
  отдельного подтверждения условий.
- Для внутренних задач допустимо извлечение проверяемых утверждений на основе
  сохранённого снимка.
- Атрибуция и правовой статус: `knowledge/source-attribution.md`.
