# Карточка источника: `BSIM`

## Идентификатор

`BSIM`

## Статус

- Состояние учёта: учтён
- Состояние получения: `external_reference` — локальная копия не хранится; отчёт
  доступен по регистрации
- Состояние пригодности: пригоден (с трением доступа)

## Что это за источник

`BSIMM` (Building Security In Maturity Model) — модель зрелости на основе
наблюдаемых практик реальных организаций; ежегодный отчёт.

## Почему источник относится к проекту

Наблюдательный ориентир зрелости для навыка `security-audit` в дополнение к
предписывающей модели `SAMM`.

  `https://www.blackduck.com/resources/analyst-reports/bsimm.html`

## Тип источника

- Тип материала: внешний справочный источник (`external_reference`)
- Тип формы: проприетарный отчёт (по регистрации)

## Условия использования

- Лицензионный статус: защищён авторским правом издателя; бесплатно по
  регистрации; открытой лицензии нет.
- Допустимое использование: краткие сведения и ссылки без крупных дословных
  фрагментов.
- Атрибуция: `knowledge/source-attribution.md`.

## Что требует осторожности

- версии обновляются (`BSIMM15`, `BSIMM16` …); указывать версию;
- описательная модель, не предписывающий стандарт; доступ требует регистрации.
