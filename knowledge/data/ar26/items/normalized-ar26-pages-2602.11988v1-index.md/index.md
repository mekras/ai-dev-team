# Evaluating AGENTS.md: Are Repository-Level Context Files Helpful?

## Ключевые положения

- Исследуется эффективность контекстных файлов (AGENTS.md и аналоги).
- Рассматриются две сценарии:
  - SWE-bench задачи с LLM-генерируемыми файлами по рекомендациям;
  - реальные issues из репозиториев с developer-written контекстом.
- В нескольких агентах и моделях найдено ухудшение success rate при
  использовании context files относительно отсутствия репозиторного контекста.
- При этом возрастает стоимость инференса более чем на 20%.
- Контентные файлы повышают поведение исследования (широкая проверка,
  более глубокая навигация), и агенты в целом следуют закладкам инструкций.
- Вывод: требования должны быть минимальными и не перегружать контекст.
- Вступление указывает, что практику используют многие репозитории
  (около 60k по состоянию на публикацию).
