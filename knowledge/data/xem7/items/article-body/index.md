# Локальный артефакт: `XEM7/article-body`

## Назначение

Этот файл не содержит полный текст связанной X Article.

Он нужен как отслеживаемый манифест для корпуса знаний: показывает, что полный
текст статьи был получен локально, но не включён в Git из-за лицензионных и
публикационных ограничений источника.

## Что хранится локально

- Тип локального артефакта: текстовый снимок тела статьи
- Дата сохранения: `2026-06-01T09:46:19Z`
- Источник: `https://x.com/i/article/2060729909690224640`
- Происхождение: снимок предоставлен человеком-заказчиком для внутренней работы
  корпуса
- SHA256 локального полного снимка:
  `415dadb13d8de63642a0f4372f96796895a06d1afe35f73e792485479f9ee3aa`
- Объём локального снимка: `223 lines`, `3100 words`, `18304 bytes`

## Ограничения

- Полный текст статьи не должен попадать в Git этого публичного репозитория без
  отдельного правового основания.
- Этот файл не заменяет локальный полный снимок и не должен использоваться как
  его содержательный пересказ.
- Для проверки утверждений по источнику нужно сверяться с локальным снимком,
  который хранится вне Git.
