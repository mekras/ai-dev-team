## Назначение

Утверждения фиксируют структуру и назначение `OWASP WSTG` для навыков
`security-audit` и `security-tooling`. Источник — `external_reference`; копия не
хранится. Статус — `ready_for_review`. Лицензия — `CC BY-SA 4.0` (атрибуция).

### WSTG-001. Назначение руководства

- Утверждение: WSTG — открытое руководство по тестированию безопасности
  веб-приложений и сервисов: методология встраивания тестирования в жизненный
  цикл и каталог тестов с устойчивыми идентификаторами
  `WSTG-<категория>-<номер>`.
- Фрагмент источника: обзор проекта.
- Область применения: специалист по безопасности.

### WSTG-002. Категории тестирования

- Утверждение: тесты сгруппированы по категориям: сбор информации, конфигурация
  и развёртывание, управление идентичностью, аутентификация, авторизация,
  управление сессиями, проверка ввода, обработка ошибок, криптография,
  бизнес-логика, клиентская сторона, тестирование API.
- Фрагмент источника: структура категорий.
- Область применения: специалист по безопасности, разработчик.

### WSTG-003. Роль в аудите

- Утверждение: WSTG применяется как методическая основа тестирования и пентеста
  в навыке `security-audit`; конкретный тест берётся из первоисточника по
  идентификатору `WSTG-*`, версия сверяется по каноническому адресу.
- Фрагмент источника: назначение руководства.
- Область применения: специалист по безопасности, роль проверки результата.
