## Назначение

Утверждения фиксируют перечень и назначение `OWASP Top 10` (редакция 2021) для
навыков `security-threat-modeling` и `security-audit`. Источник —
`external_reference`. Статус — `ready_for_review`. Лицензия — `CC BY-SA 4.0`.

### OWTT-001. Назначение перечня

- Утверждение: OWASP Top 10 — перечень наиболее критичных рисков безопасности
  веб-приложений для повышения осведомлённости, а не полный стандарт требований.
- Фрагмент источника: обзор проекта.
- Область применения: специалист по безопасности, разработчик.

### OWTT-002. Категории редакции 2021

- Утверждение: редакция 2021 включает A01 Broken Access Control, A02
  Cryptographic Failures, A03 Injection, A04 Insecure Design, A05 Security
  Misconfiguration, A06 Vulnerable and Outdated Components, A07 Identification
  and Authentication Failures, A08 Software and Data Integrity Failures, A09
  Security Logging and Monitoring Failures, A10 Server-Side Request Forgery
  (SSRF).
- Фрагмент источника: перечень категорий.
- Область применения: специалист по безопасности, разработчик.

### OWTT-003. Связь с дизайном и таксономиями

- Утверждение: категория A04 «Insecure Design» подчёркивает моделирование угроз
  и безопасное проектирование; категории связаны со слабостями `CWE`; для
  проверяемых требований используется `ASVS`.
- Фрагмент источника: описание категорий и связей.
- Область применения: специалист по безопасности, архитектор.

### OWTT-004. Изменчивость редакций

- Утверждение: перечень обновляется раз в несколько лет, состав и порядок
  категорий меняются — при использовании указывать редакцию и сверяться с
  актуальной по каноническому адресу.
- Фрагмент источника: история редакций.
- Область применения: специалист по безопасности.
