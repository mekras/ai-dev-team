# Custom instructions в VS Code (нормализованное представление)

## URL

- `https://code.visualstudio.com/docs/agent-customization/custom-instructions`

## Ключевое содержимое

- Custom Instructions — механизм передачи правил для чатов и обработки задач.
- Инструкции можно применять всегда (always-on) или условно (по файлам,
  типу задачи, паттернам).
- Поддерживаемые механизмы:
  - `.github/copilot-instructions.md`
  - `AGENTS.md`
  - `.instructions.md`
  - `CLAUDE.md`
  - уровни организации (project/workspace/user).
- Несколько файлов для workspace могут комбинироваться.
- Inline suggestions (встроенные автодополнения в редакторе) в базовом
  потоке инструкциями не затрагиваются.

## Особенности применения

- При нескольких файлах порядок применения в контексте чата нефиксированный.
- Для точной прицелки можно задавать instruction file locations для
  отдельных файлов и подкаталогов.
- Предпочтение гибкости через scoped инструкции для конкретных областей
  репозитория.
