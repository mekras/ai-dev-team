## О чём текст

- AGENTS.md позиционируется как "README for machines" для AI-ассистентов.
- Признаётся широкое распространение подхода и рост формализации
  (сотрудничество нескольких вендоров).
- Основной фокус: технический контекст для моделей в виде команд,
  структур, gotchas и шаблонов.

## Концепты

- Выделяются разделы:
  - Что такое AGENTS.md и зачем он нужен.
  - Нынешние стандарты структуры и naming.
- Практика file layout:
  - конкретные разделы по технологиям и пайплайну.
- Рекомендуются шаблоны:
  - компактный для простых проектов,
  - расширенный для сложных стеков.
- Значимое внимание безопасности:
  - секции "never include secrets".
  - правила про переменные окружения, критические ошибки.
