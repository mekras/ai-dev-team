# Корпус знаний проекта

Корпус знаний проекта, или просто «корпус знаний», это главный источник истины
для агентов в этом проекте.

## Что такое корпус знаний

Корпус знаний — это набор подключённых источников и производных материалов,
которые проект считает опорными для работы агентов.

В него входят материалы, на которые агенты должны опираться при выполнении
задач, проверке изменений и подготовке новых результатов.

## Зачем нужен корпус знаний

Корпус знаний нужен, чтобы агенты:

- работали не по случайному контексту задачи, а по зафиксированным источникам;
- опирались на одинаковые основания в разных задачах;
- могли проверять, изменился ли внешний источник;
- не теряли важные для проекта правила, решения и ограничения;
- обновляли производные материалы на основании изменений в источниках.

## Как с ним работать

Основная работа с корпусом знаний состоит в управлении источниками.

Обычно нужно:

1. Добавлять в корпус новые источники, если они должны использоваться агентами.
2. Запускать проверку источников, чтобы понять, появились ли изменения.
3. Разбирать результаты проверки и решать, нужно ли обновлять производные
   материалы проекта.
4. Удалять или заменять источники, если они больше не должны входить в корпус.

Добавлять в корпус стоит только те источники, которые действительно должны
служить основанием для дальнейшей работы агентов.
