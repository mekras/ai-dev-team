# Бэклог продукта

Этот файл фиксирует открытые вопросы и задачи по развитию продукта, которые ещё
не решены и не запланированы в конкретную версию. Записи здесь — повод для
решения владельца, а не принятое решение.

## Открытые вопросы

### Развитие слоя согласования понятий

Введён слой согласования понятий `knowledge/concepts.yml` (решение —
`docs/adr/0001-concepts-reconciliation-layer.md`), пока в виде MVP с несколькими
засеянными понятиями. Осталось:

- проверка целостности ссылок слоя (`defined_by`, `covered_in`) в
  `kc-validation`, чтобы слой не дрейфовал; сейчас это вне продукта (валидатор в
  коллекции корпуса);
- правило маршрутизации «вопрос о понятии → сверься со слоем» в `ait-routing`;
- расширение охвата по требованию (новые сквозные понятия и коллизии);
- решение, нужен ли отдельный продуктовый канон терминов (связка с `ru-lang`)
  поверх корпусного слоя.
