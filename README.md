# Команда агентной разработки ПО

Команда агентов ИИ, которую можно подключить к проекту по
разработке ПО. Включает в себя необходимые роли навыки и правила
работы, проверки качества и согласованный порядок принятия решений.
Обеспечивает стабильный уровень качества за счёт системного
применения накопленных практик и снижает риски, вызванные человеческим
фактором.

## Подключение к проекту

1. Установите продукт.

```shell
npm install -g git+https://github.com/mekras/ai-dev-team.git
```

2. Подключите продукт к Codex.

```shell
ai-dev-team setup codex --channel stable
```

3. Проверьте подключение.

```shell
ai-dev-team-mcp --check --channel stable
```

Если глобальная установка недоступна, можно проверять так:

```shell
node ./bin/ai-dev-team-mcp.js --check --channel stable
```

4. Добавьте подключение в `AGENTS.md` вашего проекта.

```markdown
ai-dev-team agents --channel stable
```
