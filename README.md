# Команда агентной разработки ПО

Команда агентов ИИ, которую можно подключить к проекту по разработке ПО.
Включает в себя необходимые роли навыки и правила работы, проверки качества и
согласованный порядок принятия решений. Обеспечивает стабильный уровень качества
за счёт системного применения накопленных практик и снижает риски, вызванные
человеческим фактором.

## Подключение к проекту

Для подключения необходим
[Agent Package Manager](https://github.com/microsoft/apm).

Выберите клиент и выполните команды в корне целевого проекта.

Для Claude:

```shell
apm install mekras/ai-dev-team#^0.2.2 --target claude
apm compile --single-agents --target claude
```

Для Codex:

```shell
apm install mekras/ai-dev-team#^0.2.2 --target codex
apm compile --single-agents --target codex
```

APM добавит продукт в зависимости текущего проекта, установит локальную копию
продукта и его зависимости, затем создаст клиентские инструкции для выбранной
цели. В скомпилированных инструкциях появится объявление подключения:

```markdown
Проект использует `ai-dev-team`.
```

Не опускайте `--target`: проект нужно подключать явно либо для `claude`, либо
для `codex`.

Проверкой подключения считается наличие продукта в зависимостях проекта и
успешная компиляция инструкций для выбранной цели.

## Зависимости разработки продукта

При разработке самого `ai-dev-team` зависимые коллекции навыков подключаются
через `APM`.

В корне этого репозитория выполните:

```shell
apm install --target agent-skills --frozen
npm install
```

Если нужно обновить закреплённые зависимости и lockfile, выполните:

```shell
apm deps update --target agent-skills
```

## Начальная настройка проекта

Для настройки нового проекта или перенастройки существующего проекта можно
использовать навык [ait-setup](team/skills/ait-setup/SKILL.md). Откройте ваш
инструмент для общения с агентами и дайте команду вида:

```text
Настрой проект.
```
