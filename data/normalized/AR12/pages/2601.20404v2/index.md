# The Impact of AGENTS.md Files on the Efficiency of AI Coding Agents

## Содержание статьи

- Исследуется влияние `AGENTS.md` на эффективность AI-агентов.
- База эксперимента: 10 репозиториев и 124 pull request.
- Сравнение двух режимов:
  - с `AGENTS.md`
  - без `AGENTS.md`.
- Метрики: wall-clock runtime и output token consumption.
- Объективные результаты по метрикам:
  - снижение медианного времени выполнения примерно на `28.64 %`;
  - снижение расхода токенов примерно на `16.58 %`;
  - сопоставимое поведение по завершению задач.
- Авторы выделяют практические последствия для деплоймента агентов и
  формулируют направление дальнейших исследований по влиянию контекста.
