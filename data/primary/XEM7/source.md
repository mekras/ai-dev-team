# Источник: `XEM7`

## Статус

Источник принят в корпус знаний как внешний публичный веб-материал с
ограничениями на хранение исходных материалов в Git.

## Происхождение

- Платформа: `X`
- Автор: `EXM7777` / `Machina`
- Страница поста: `https://x.com/EXM7777/status/2060736517564477901`
- Связанная статья: `https://x.com/i/article/2060729909690224640`
- Дата получения поста: `2026-06-01T09:26:10Z`
- Дата получения связанной статьи: `2026-06-01T09:26:44Z`
- Дата сохранения тела статьи: `2026-06-01T09:46:19Z`
- Способ получения HTML: HTTP GET через настроенный прокси с локальным
  сохранением HTML и заголовков ответа
- Способ получения тела статьи: текстовый снимок статьи, предоставленный
  человеком-заказчиком

## Что сохранено

Для обработки локально получены следующие первичные артефакты:

- очищенный HTML-снимок поста:
  `data/primary/XEM7/pages/status/index.html`
- очищенные заголовки ответа поста:
  `data/primary/XEM7/pages/status/response-headers.txt`
- очищенный HTML-снимок связанной статьи:
  `data/primary/XEM7/pages/article/index.html`
- очищенные заголовки ответа связанной статьи:
  `data/primary/XEM7/pages/article/response-headers.txt`
- тело связанной статьи:
  `data/primary/XEM7/pages/article-body/index.md`
- индекс сохранённых страниц:
  `data/primary/XEM7/page-index.tsv`

В Git сохраняются только метаданные источника, индекс с хэшами, карточка
источника, пересказ, извлечённые утверждения и производные выводы без крупных
дословных фрагментов.

Полные HTML-снимки, заголовки ответа, полный текст статьи и полное
нормализованное представление исключены из Git через `.gitignore` и используются
как локальные первичные данные для проверки.

## Очистка перед сохранением

X вернул в ответах cookie и inline-скрипт установки cookie. Эти значения не
сохраняются в корпусе.

Перед записью в отслеживаемый слой были удалены:

- значения `set-cookie` в заголовках ответа;
- содержимое inline-скрипта `document.cookie` в HTML.

Остальное содержимое HTML-снимков сохранено без смысловой переработки.

## Условия использования

- Открытая лицензия на повторное использование поста и связанной статьи не
  установлена.
- Правообладатель пользовательского контента сохраняет права на опубликованный
  материал, если применимые условия платформы не задают иное.
- Условия платформы X не являются разрешением для этого репозитория на
  публикацию полного текста или крупных дословных фрагментов.
- Уровень уверенности: средний; открытая лицензия в сохранённых снимках не
  выявлена.
- Хранение полного текста статьи в Git публичного проекта: не допускается без
  отдельного правового основания.
- Локальное хранение полного текста статьи: допустимо как ограниченный первичный
  снимок для внутренней обработки.
- Тело статьи получено как текстовый снимок, предоставленный
  человеком-заказчиком для внутренней работы корпуса.
- Допустимое использование в корпусе: внутренний анализ, извлечение коротких
  проверяемых утверждений и ссылки на сохранённые артефакты.
- Публичное воспроизведение полного текста или крупных дословных фрагментов
  требует отдельного правового основания.
- Атрибуция и правовой статус: `data/source-attribution.md`.

## Ограничения

- Снимки отражают состояние страниц только на момент получения.
- HTML X является клиентским приложением и содержит большой объём служебного
  кода.
- Снимок поста содержит объект поста в `window.__INITIAL_STATE__`.
- Пост содержит ссылку на X Article, но не содержит текст статьи.
- Прямой снимок X Article не содержит тело статьи в `window.__INITIAL_STATE__`
  при неавторизованном получении.
- Тело X Article сохранено отдельным локальным текстовым снимком и может
  использоваться для нормализации и извлечения утверждений.
- Дальнейшее смысловое извлечение возможно только из данных, реально сохранённых
  локально и зафиксированных в индексе.

## Назначение в проекте

Источник может использоваться как первичный материал после отдельной проверки:

- для учёта самого факта публикации и ссылки автора;
- для извлечения утверждений из сохранённого тела связанной статьи;
- для анализа влияния после нормализации сохранённого тела статьи.
