## Что сохранено

Для обработки локально получены следующие первичные артефакты одной единицы
источника:

- `data/primary/PPRX/pages/designing-refining-maintaining-agent-skills/response-headers.txt`

В Git сохраняются только метаданные источника, индекс с хэшами, карточка
источника, извлечённые утверждения и производные выводы без крупных дословных
фрагментов.

Полный HTML-снимок и полное нормализованное представление исключены из Git через
`.gitignore` и используются как локальные первичные данные для проверки.

## Условия использования

- Открытая лицензия на повторное использование материалов `Perplexity Research`
  не установлена.
- На источник распространяются условия Perplexity, если для отдельной единицы
  источника не указано иное.
- Уровень уверенности: средний; отдельная открытая лицензия в сохранённом снимке
  не выявлена.
- Хранение полного HTML-снимка в Git публичного проекта: не допускается без
  отдельного правового основания.
- Локальное хранение полного HTML-снимка: допустимо как ограниченный первичный
  снимок для внутренней обработки.
- Допустимое использование в корпусе: внутренний анализ, извлечение коротких
  проверяемых утверждений и ссылок на сохранённые артефакты.
- Публичное воспроизведение значительных фрагментов, публикация полного текста
  или создание производных материалов с крупными дословными заимствованиями
  требует отдельной проверки правового основания.
- Атрибуция и правовой статус: `data/source-attribution.md`.

## Ограничения

- Источник сейчас представлен одной единицей: статьёй
  `Designing, Refining, and Maintaining Agent Skills at Perplexity`.
- Это снимок состояния страницы на дату получения, а не постоянная ссылка на
  текущее состояние сайта.
- Сырой HTML может содержать навигацию, служебные данные, встроенные скрипты и
  оформление сайта.
- Нормализованные представления и извлечённые утверждения должны строиться
  только из локально сохранённых артефактов соответствующей единицы источника.
- Материалы источника не являются принятыми решениями текущего проекта.

## Назначение в проекте

Этот источник может использоваться как первичный материал для:

- изучения практик проектирования и сопровождения навыков агентов;
- извлечения проверяемых утверждений о структуре, маршрутизации и сопровождении
  навыков;
- анализа влияния на переносимый слой `skills/` с учётом лицензионных
  ограничений.
